# MTech Solutions Enterprise Website

This project will contain a production-grade, enterprise-level website for MTech Solutions, built with React + Vite on the frontend and Node.js + Express with MySQL on the backend.

## Folder Structure
```
mtech-solutions/
├── client/          # React + Vite frontend
└── server/          # Node.js + Express backend
```

## Initial Setup
1. Clone this repository
2. Create a `.env` file in the root with necessary environment variables (e.g., `DB_HOST`, `PORT`)
3. Run `npm install` in both `client` and `server` directories
4. Set up MySQL database and run schema migrations

## Design Goals
- Modern, premium, responsive UI with glassmorphism accents
- Dark mode with persistent toggle
- SEO-optimized and WCAG-compliant
- Full CI/CD pipeline for VPS/Hostinger/AWS/DigitalOcean deployment
